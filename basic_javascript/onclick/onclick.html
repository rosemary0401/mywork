External JavaScript

1.JavaScript files have the file extension .js.

2.To use an external script, put the name of the script file in the src (source) attribute of a script tag

3.External scripts cannot contain script tags.

4.You can place an external script reference in head or body as you like.

5.To add several script files to one page - use several script tags

6.External scripts can be referenced with a full URL or with a path relative to the current web page.
