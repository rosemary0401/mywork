遊子思鄉味ー台式滷肉飯

April 27, 2020

台灣的滷肉飯做法簡單，材料只有豬肉、紅蔥頭及醬油等調味料，細火慢滷後就成色香味俱全超級下飯的料理。旅外的家人想吃台灣味的時候，台式滷肉飯最能滿足思鄉之味。

十三香蔬果牛肉麵ー辛香料與蔬果熬煮而成的牛肉麵

April 26, 2020

十三香包括花椒、八角、山奈、白芷、高良薑、草果、肉豆蔻、小茴香、砂仁、木香、桂皮、甘薑、丁香等十三種香料，比例以花椒、八角最多，再來是山奈、白芷、高良薑，而草果、肉豆蔻、小茴香、砂仁、木香、桂皮、甘薑、丁香則各一點點而己。 辛辣的部分用到蔥、薑、蒜及辣椒。蔬菜有紅、白蘿蔔、洋蔥、蕃茄，水果有蘋果。
