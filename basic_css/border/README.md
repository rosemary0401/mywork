# Explain
* `margin` property has four individual margin properties:
  - `margin-top`
  - `margin-right`
  - `margin-bottom`
  - `margin-left`
* `margin` properties can have different values:
  - margin: auto
  - margin: top right bottom left
  - margin: top&bottom right&left
  - margin: top right&bottom bottom
  - margin: inherit
